Block, Inline, and Inline-Block Elements

Block Elements

This is a block element.
Block elements take up the full width.
Each block starts on a new line.

Inline Elements

This is an inline element. Inline elements stay in the same line. They only take up as much width as needed.

Inline-Block Elements

This is an inline-block element.
It acts like a block element.
But stays inline like an inline element.
